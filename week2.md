>![image](https://github.com/user-attachments/assets/3ab7aa90-5515-4dbe-b2b2-b6ed10fe80a9)


```
조건 -> WHERE
추출 -> SELECT
요약 -> GROUP BY, GROUP BY ALL(컬럼 여러개일때 컬럼 전부 나열 안하고 group by 가능)
집계 -> AVG, COUNT, COUNTIF, SUM, MAX, MIN
   -> HAVING (GROUP BY 후)
정렬 -> ORDER BY DESC
제한 -> LIMIT
```
